# Tarjeta intérpretes

## Descripción general

El componente **`List`** representa una tarjeta de intérprete con foto, nombre y descripción.  
Puede marcarse como destacado mediante la propiedad `esNota10`, lo que cambia su estilo visual.

## Accesibilidad

El componente `List` está diseñado con enfoque en la accesibilidad:

- **`tabIndex="0"`:** Permite que la tarjeta sea enfocada y navegable.  
- **`aria-label`:** Proporciona un resumen verbalizado claro.  
- **`sr-only`**: Se utiliza para ocultar la biografía visualmente.
